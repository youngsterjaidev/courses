HTML

What is HTML

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content."Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

CSS

What is CSS

CSS (Cascading Style Sheets) is the code that styles web content.Like HTML, CSS is not a programming language. It's not a markup language either. CSS is a style sheet language. CSS is what you use to selectively style HTML elements.

GIT

What is GIT

Git is a free and open source distributed version control system that allows you to track and manage changes to your code projects. It is designed to handle everything from small to very large projects with speed and efficiency. Git is easy to learn and has a tiny footprint with lightning fast performance¹. It also supports features like cheap local branching, convenient staging areas, and multiple workflows.

left right

GitHub

What is GitHub ?

GitHub is a code hosting platform that works with Git to provide collaboration and version control features. GitHub lets you (and others) work together on projects, manage your Git repositories, review code, track bugs and features, power your CI/CD and DevOps workflows, and secure code before you commit it. GitHub also offers various tools and services to help developers, such as GitHub Copilot, GitHub Actions, GitHub Mobile, GitHub Discussions, GitHub Sponsors, Secret scanning, Dependabot, and Code scanning¹..
